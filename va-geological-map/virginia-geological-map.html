Virginia geologic map data

A GIS database of geologic units and structural features in Virginia, with lithology, age, data structure, and format written and arranged just like the other states. [Legend for lithology colors] Show in Google Earth or download KML:
  • vageol.kml (14.6M bytes)
  • vageol.kmz (5.1M bytes)
Show in your GIS using OGC WMS:
http://mrdata.usgs.gov/services/va?service=WMS&version=1.1.1&request=GetCapabilities& [Instructions]
or OGC WFS:
http://mrdata.usgs.gov/services/va?service=WFS&version=1.0.0&request=GetCapabilities& [Instructions]
Use lithology_symbology.lyr ArcGIS layer file to symbolize lithology when using WFS

Geological attributes: VAcsv.zip CSV format
Geologic units: VAgeol_dd.zip Shapefile, polygon

Conservation Lands Database 

Shapefile

 

Tags

 

environment, boundaries, planning, biota 

 

Summary

 

Statewide GIS coverage of Conservation Lands in Virginia to serve as a land conservation planning tool. 

Description

 

This dataset contains the boundaries for lands of conservation and recreational interest in Virginia. The 

Conservation Lands Database is constantly being edited and updated. Data is released to the public 
quarterly and posted to the download section of the website: 

http://www.dcr.virginia.gov/natural_heritage/cldownload.shtml 

Conservation Easement boundaries must be requested directly via email: david.boyd@dcr.virginia.gov
